## 🚀 Quick Start (3 Steps)

### Step 1: Install Dependencies
```bash
flutter pub get
flutter pub run build_runner build --delete-conflicting-outputs
```

### Step 2: Add API Key
Edit `lib/features/chat/providers/ai_provider.dart`:
```dart
static const String _apiKey = 'YOUR_GEMINI_API_KEY';
```

### Step 3: Add Fonts
Download Poppins and add to `assets/fonts/`:
- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf

### Run!
```bash
flutter run
```
